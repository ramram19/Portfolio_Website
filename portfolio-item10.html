SchoolProject

Ryders Boardshop

This was a school project for a made up company. Luckily, I already made a logo for my other class. I thought this would be a great way to create some cohesion between the two classes and make this a brand.

The main project for my Indesign class was to create a brochure for a company. We had free reign to choose whatever direction we wanted to go in. The bulk of this project was to develop a layout for a trifold brochure and I kind of went overboard with this project. I wanted to put myself to the test and see what I could come up with.

This was another one of those projects where I thought nothing good was going to come out of it, but I think it came out really well done.
