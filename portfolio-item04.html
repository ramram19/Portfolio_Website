Mitosis Illustration

This was a project given to us to create an illustration for the process of mitosis. I already had somewhat of an idea I wanted this project to go in. Which was grunge and to use textures to give it that extra oompf to set it over the top. I didn't want to get distracted with this idea and I explored many options on how I wanted to style it. I really wanted to use Photoshop in conjuction with Illustrator to show I can use both programs to create a project.
